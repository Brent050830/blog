  • error

try:
  print(name)
except NameError as e:
  print("出现了变量未定义的异常")
  print(e)# 将错误的说明语句输出：name 'name' is not defined
# 只捕获nameerror
# 捕获多种异常：括号，类似元组的形式
try:
  print(name)
  1/0

# 捕获所有异常：最开始的语法就可以
except Exception as e:# 或利用这种！Exception是一个顶级的异常

模块的导入

  • 一个Python文件，有函数、类、变量
  • 将模块当做工具包使用

语法：

as
